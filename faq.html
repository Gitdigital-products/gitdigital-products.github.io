Frequently Asked Questions

Find answers to common questions about the Solana KYC SDK, integration, compliance, and more.

What is the Solana KYC SDK? +

The Solana KYC SDK is a software development kit that enables Solana blockchain applications to implement Know Your Customer (KYC) and identity verification processes. It provides a seamless way to verify user identities while maintaining compliance with global regulations.

Key features include:

  • Document verification (passport, driver's license, ID cards)
  • Facial recognition and liveness detection
  • On-chain verification status storage
  • Support for multiple verification tiers
  • Global compliance coverage
How does it differ from traditional KYC solutions? +

Traditional KYC solutions are often centralized, slow, and not designed for blockchain applications. Our SDK is:

  1. Blockchain-native: Built specifically for Solana with wallet integration
  2. Privacy-focused: Uses zero-knowledge proofs and minimal data storage
  3. Fast: Verification completes in seconds vs. days with traditional providers
  4. Cost-effective: Lower fees due to blockchain efficiency
  5. Interoperable: Verification status can be shared across dApps
How long does integration take? +

Most developers can integrate the basic KYC flow in under 30 minutes. The complete integration with custom UI and advanced features typically takes 1-2 days.

Basic integration timeline:

  • 0-15 minutes: Install SDK and get API key
  • 15-30 minutes: Implement basic verification flow
  • 30-60 minutes: Test with development environment
  • 1-2 hours: Customize UI and add advanced features

We provide comprehensive documentation, example projects, and template code to accelerate integration.

Which wallets are supported? +

We support all major Solana wallets:

  • Phantom (Full support with automatic connection detection)
  • Solflare (Complete integration including hardware wallet support)
  • Ledger (Hardware wallet integration for maximum security)
  • Backpack
  • Glow
  • Any wallet supporting Solana's Wallet Standard

The SDK automatically detects the connected wallet and adapts the verification flow accordingly. You can also manually specify wallet preferences in the configuration.

Can I customize the verification UI? +

Yes, the SDK provides multiple customization options:

1. Theme customization: Change colors, fonts, and styling to match your brand

2. Layout options: Choose between modal, inline, or full-page verification flows

3. Custom components: Replace specific UI elements with your own components

4. Localization: Support for multiple languages and regional requirements

5. CSS overrides: Complete control over styling via CSS classes

For most use cases, basic theme customization is sufficient. For advanced customizations, we provide React and Vue component libraries.
